This page has the ludorum-risky library loaded. Open your browser Javascript console to test. In Windows use: Ctrl+Shift+J in Chrome, Ctrl+Shift+K in Firefox, F12 in Internet Explorer, Ctrl+Shift+I in Opera.

Reference

ludorum_risky Namespace of the library at window level.
var MAP = ludorum_risky.maps.classic; MAP; Classic map to use with ludorum_risky.Risk.
var GAME = new ludorum_risky.Risk({ boardgame: MAP }); GAME; Implementation of Risk-like games as a subclass of ludorum.Game. The constructor can be used like this: new ludorum_risky.Risk({ boardMap: ludorum_risky.MAPS.classic, rounds: 100 }).
var PLAYER = new RiskContinentPlayer(); PLAYER; Automatic players implementation, with a priority on continent conquest.
